## üé® **Screenshot Best Practices**

### **Visual Guidelines**:
- **Clean UI**: Close unnecessary browser elements
- **Good Lighting**: Use light browser theme for clarity
- **Readable Text**: Ensure all text is crisp and readable
- **Professional Context**: Use clean, professional websites as background
- **Consistent Branding**: Show extension's visual identity

### **Technical Setup**:
1. **Browser Setup**:
   - Use Chrome with clean profile
   - Hide bookmarks bar if cluttered
   - Use light theme for better contrast
   - Set zoom to 100%

2. **Extension Setup**:
   - Create an attractive pet (colorful, clear)
   - Generate realistic tab scenarios
   - Ensure all features are working

3. **Screen Capture**:
   - Use built-in screenshot tools or Snagit
   - Capture at high resolution
   - Crop precisely to required dimensions
   - Save as PNG for best quality
